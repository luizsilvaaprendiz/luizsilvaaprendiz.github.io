Introdução ao HTML

O que é HTML?

HTML é a linguagem de marcação padrão para criar páginas da Web.

HTML significa Hyper Text Markup Language.

HTML descreve a estrutura de uma página da Web.

HTML consiste em uma série de elementos.

Elementos HTML informam ao navegador como exibir o conteúdo.

Elementos HTML rotulam partes do conteúdo como "este é um título", "este é um parágrafo", "este é um link", etc.

Tags e elementos HTML

Tag de abertura é formada por letras, palavras-chave completas ou abreviadas, que são colocados entre os sinais MENOR QUE (<) e MAIOR QUE (>).

Já a Tag de fechamento segue a mesma estrutura anterior seguida de BARRA (/) logo após o símbolo MENOR QUE (<).

Por isso, o conjunto de uma TAG de abertura com um conteúdo qualquer e uma TAG de fechamento é chamado de elemento HTML.

Observações:

1.TAGs de autofechamento não têm conteúdo e servem, por exemplo, para indicar uma quebra de linha ou pular uma linha etc.

2. A instrução INDEX.HTML só funciona no servidor e não em arquivo local.

3. As expressões página, documento e arquivo são sinônimas.

4. O computador não identifica os elementos da organização do texto da mesma forma que os seres humanos.
